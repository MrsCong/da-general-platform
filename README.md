# da-general-platform
# dragon通用平台 提供以下组件：
    # dgp-common : 提供通过工具类
    # dgp-core : 提供基础架构封装
    # dgp-web : 提供web封装服务
    # dgp-mybatis: 提供mybatis服务
    # dgp-elasticsearch: 提供elasticsearch封装服务
    # dgp-kafka: 提供kafka封装服务
    # dgp-redis: 提供redis封装服务
    # dgp-mq: 提供基于easyexcel封装的excel服务
    # dgp-job: 提供基于xxl-job的封装的分布式任务服务
